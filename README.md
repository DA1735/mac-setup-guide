# 数学科大学院生のための Mac セットアップガイド

本リポジトリは、数学科の大学院生・研究者が貸与されたMac（または個人のMac）を、研究・計算・論文執筆において「**最も効率的かつクリーンな状態**」にセットアップするための公式マニュアル集です。GUIで適当にマウス操作をして設定するのではなく、ターミナルを用いた再現性の高いモダンな構築手順を体系化しています。

## 🎯 設計方針とベストプラクティス

本ガイドでは、環境汚染（システムが壊れたり、他人の環境で動かなくなること）を防ぐため、以下の原則を徹底しています。

1. **パッケージマネージャーの一元化**
   アプリやツールの導入は、手動ダウンロードを極力避け、コマンド1行で管理できる **Homebrew** を推奨します。
2. **本番環境と同一の計算環境（Docker First）**
   データ分析やプログラミング環境は、Mac本体を汚さない **Docker** コンテナ（`jupyter/datascience-notebook` 等）の利用を最優先とします。
3. **ローカル環境のモダン化**
   どうしてもローカルに直接言語を入れる場合、レガシーなAnacondaなどは例外なく非推奨とし、Pythonには `uv`、R言語には `rig` / `rv` を採用して高速・クリーンな環境を作ります。
4. **モダンな執筆環境**
   LaTeXの執筆は、VS Codeと `latexmk` を組み合わせた「保存時自動コンパイル・自動整形」を採用し、執筆体験（DX）を最大化します。

### 📌 2. リファレンス・詳細マニュアル
特定の設定を極めたい時や、個別機能を使いたい際に参照する応用マニュアルです。

* [プリンター設定マニュアル（Mac）](docs/05-printer.md)
  * 学内プリンター室への物理LAN経由の直接接続・ドライバー設定を解説。
* [数学科大学院生のためのLaTeX完全ガイド (Mac編)](docs/06-latex-advanced.md)

### 📌 3. テンプレート (`docs/templates/`)
研究ですぐに使えるファイル群です。
* `template-uplatex.tex` : 日本語レポート・論文の標準テンプレート（uplatex対応）
* `template-beamer.tex` : プレゼンテーション用スライドテンプレート（Beamer）
* `template-reference.tex` / `myref.bib` : BibTeX を用いた参考文献付きの論文テンプレート
* `test_ruff.py` : Pythonの自動整形設定（Ruff）を確認するためのテストコード
