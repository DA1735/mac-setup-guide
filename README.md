## 本ガイドの基本方針
このガイドでは、アプリケーションの管理を可能な限り **Homebrew** というパッケージ管理システムで行うことを推奨しています。GUIのインストーラを手動でダウンロードするのではなく、コマンド一行でインストール・更新を行う習慣を身につけることで、将来的な環境移行や管理をスムーズにします。
