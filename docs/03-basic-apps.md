# 03. 基本アプリのインストール (Google Chrome 等)

数学科での研究生活や、各種学内システム（Google Workspace）を使うために基本アプリを Homebrew でインストールします。

## Google Chrome
大学の連絡手段や各種システムが Google アカウントに紐付いているため推奨します。

```bash
brew install --cask google-chrome
```

## その他コミュニケーションツール・アプリ
オンラインミーティングや連絡のために、必要に応じて以下のツールもインストールしてください（任意）。

- **Skype / Slack / Zoom**: 
  ```bash
  brew install --cask slack zoom
  ```
- **DeepL**: 論文の翻訳に便利です。
  ```bash
  brew install --cask deepl
  ```

※ 数学の論文執筆に欠かせない **LaTeX環境やエディタ（VS Code）の導入については、次のステップ（全員必須）で行います**。
