# 01. 初期設定 (Hello 画面から)

Mac の電源を最初に入れると「Hello」と表示された多言語の画面が始まります。このステップは各自の好みに合わせて進めて問題ありませんが、推奨事項をいくつか挙げます。

## 基本的な進め方
1. **言語と地域**: 「日本語」「日本」を選択します。
2. **アクセシビリティ**: 必要に応じて設定しますが、通常はそのまま続けます。
3. **Wi-Fi ネットワーク**: 学内（または自宅）の安全な Wi-Fi に接続します。
   - ※ セットアップ中に大きなデータのダウンロードが発生するため、安定したインターネット接続が必要です。
4. **データの移行**: 別の Mac から移行する場合は「移行アシスタント」を使いますが、今回は新規セットアップとして「今は行わない」を選択することを推奨します。
5. **Apple ID**: 普段使っているものがあればサインインします。
6. **コンピュータアカウントの作成**: 
   - **フルネーム**: 本名（漢字等）を入力。
   - **アカウント名**: **英小文字と数字のみ**を推奨します（例: `suzuki`）。ディレクトリ名として使われるため、日本語やスペースは避けてください。
7. **Touch ID / Apple Pay**: 必要に応じて設定します。
8. **外観モード**: ライトモード、ダークモード、自動から好みのものを選びます。

## デスクトップが表示されたら
インターネット（Wi-Fi）に繋がっていることを確認してください。次にブラウザを開いて、このガイドの続きを確認します。
